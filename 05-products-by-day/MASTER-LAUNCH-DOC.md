## HOW TO USE THIS DOC

For every pin below, you have:
- **Image filename** (in `04-sample-pins/`)

- **Description** — paste directly into Pinterest's Description field
- **Link** — Amazon URL (use SiteStripe to get your affiliate link before scheduling)
- **Board** + **Time** — where and when to schedule
- **Product validation** for items where I couldn't verify live; check 4.4+ stars, 200+ reviews, Prime, before grabbing affiliate link

For SEED PINS — no affiliate link needed. They calibrate your boards before product pins start.

3. Open Pinterest (or Blotato)
4. Create new pin → upload image → paste title → paste description (replacing affiliate URL placeholder if needed) → set destination URL → select board → schedule for time
5. Repeat for next pin

**Time per day:** ~25 minutes for 5 pins after the first day's learning curve.
