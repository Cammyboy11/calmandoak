## TAG ROTATION REMINDER

Pinterest's algorithm penalizes identical hashtag sets across pins (looks like spam). For each pin's 5-7 hashtags:
- Always include 1 of: `#japandi` `#warmminimalism` `#calmhome`
- Always include 1 board-specific tag from the pool above
- Always include 1 product-specific tag (e.g., `#seagrassbasket` `#stonewarevase`)
- Vary the remaining 2-3 tags

Avoid using the same 5 tags on consecutive pins — Pinterest reads that as low-quality content.

---

## DESCRIPTION LENGTH GUIDE

**Sweet spot:** 250-450 characters per description.
- Under 150 chars = Pinterest deprioritizes thin content
- Over 500 chars = truncates in feed (your CTA gets cut)
- The hook + 2 sentences + 5 hashtags + disclosure = naturally hits 280-400 chars

The descriptions above are sized correctly. If you customize, keep within range.
