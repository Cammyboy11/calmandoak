# Pinterest Editorial Pivot — Batch 1 (47 Pins, 10-Day Schedule)

**Built:** 2026-05-27
**Format basis:** Cameron's winning "Japandi Mood" pin (3.6% save rate — top 1% of platform home-design pins)
**Source assets:** 17 journal covers + 30 prints, all with serif title + thin terracotta underline + "Calm & Oak" brand mark applied via batch overlay script
**Output location:** `final pins/03-editorial-batch-1/[slug].png` — 1000×1500 Pinterest-spec PNGs

## How to use this document

1. **Find each pin's image** at `final pins/03-editorial-batch-1/[slug].png` (slug listed per pin below)
2. **Copy the title + description into Pinterest's upload form** for that scheduled slot
3. **Paste the URL exactly** — the UTM parameters track which pins drive which traffic
4. **Pin to the primary board first**, then optionally save again to "Japandi Home" (the catch-all)
5. **Use 24-hour gaps between re-pinning the same image** to other boards (Pinterest spam-flag prevention)

## The format we're testing

Every pin in this batch follows the Japandi Mood template exactly:
- **Image:** 1000×1500 portrait, editorial scene (no catalog product photography)
- **Title overlay:** restrained serif (Cormorant Garamond), bottom-left, 2-word aspirational title
- **Underline rule:** thin terracotta line beneath title, ~half title width
- **Brand mark:** "Calm & Oak" upper-right in serif italic, 85% opacity
- **Caption voice:** editorial / first-person / second-person specificity (NOT catalog product-push)
- **Hashtags:** 5-7 maximum, conceptual not transactional

If after 10 days this batch outperforms the catalog-style pins by 2-3× on save rate, we go all-in on this format for the remaining months of the action plan.
